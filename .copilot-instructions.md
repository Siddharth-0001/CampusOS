# CampusOS Copilot Configuration

## Tool Access
All tools are enabled for maximum productivity and flexibility. This includes:
- File operations (create, read, edit, delete)
- Terminal execution
- Git operations
- Browser automation
- Code refactoring and analysis
- Patch application

## Development Workflow
- Prefer practical implementation over suggestions
- Use batch operations where possible for efficiency
- Minimize scripted output during normal development
- Apply changes directly rather than just proposing them

## Platform Notes
- Windows PowerShell environment
- Express.js backend (ESM modules)
- Next.js 16+ frontend with webpack (not Turbopack)
- pnpm workspaces for monorepo management
